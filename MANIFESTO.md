# 宣言

## 关于光、关于手、关于一毫米的执念

> *用秩序建立系统，用触感表达交互，用比例形成节奏，用细节沉淀气质。*
有一些东西正在从屏幕上消失。

不是功能——功能从未如此丰富。不是效率——我们点击得比任何时代都快。消失的，是那种你触碰一个按钮时，指尖下方微微凹陷的确信感；是你拨动一个开关时，齿轮咬合传来的、不可伪造的物理承诺。

我们曾经拥有过这些。收音机的调频旋钮转过整个黄昏，每一度刻度都对应一个远方的声音。打字机的金属臂击落，在纸面上留下不可撤销的凹痕。那些器物不需要说明书来解释自己——它们的形状就是它们的语言，它们的重量就是它们的承诺。

然后屏幕来了。

屏幕带来了无限的可能，也带走了一种确定。扁平化运动将界面抹成了一片纯色的旷野：干净、高效、理性，却也无从触碰。我们学会了在没有路标的平原上快速行走，但偶尔会在某个瞬间停下来，隐约觉得少了什么——像一栋没有门把手的房子，你知道怎么进去，但进去的动作本身失去了仪式感。

Tactile UI 不是怀旧。

我们不打算把皮革纹理贴回按钮表面，也不打算用仿木纹的抽屉重建一个旧世界的拟像。那条路走过了，走到了它应该结束的地方。我们要做的，是把物理世界里那些真正有价值的东西——光线、重量、深度、反馈——重新翻译成屏幕能够讲述的语言。

翻译，而不是复刻。

当光从左上方落下，按钮的右侧应该出现一道薄薄的阴影，就像午后的阳光照在书桌上的计算器。这不是装饰，这是你大脑在毫秒之内读取"这个东西可以按下去"的方式。当开关从左滑到右，凹槽里应该有一层内阴影在呼吸，告诉你：这不是一次状态切换，这是一个物理位移，是你的手指真实地改变了什么。

我们相信，好的界面不应该让用户思考"这个能不能点"。

一个圆角恰到好处的卡片，边缘带着一圈几乎察觉不到的高光；一个滑块的阻尼，在拇指滑过的速度里暗含着"到位"的暗示；一组按钮，它们的间距不是随意的 16 像素或 24 像素，而是一个自然的呼吸节奏——这些微小的信号构成了一套无声的语法，用户不会注意到它们的存在，但会在它们缺席时感到不安。

这套语法的韵脚，是黄金比例。

φ = 1.618。这个数字不是我们选择的，是向日葵的种子排列选择的，是鹦鹉螺的螺旋选择的，是人类在千年建筑史中反复验证的"看起来刚好对了"的那个比例。我们用它来定义间距的层级、字号的阶梯、阴影的扩散、圆角的递进、动效的时长。不是为了制造某种数学优越感，而是因为当所有组件共享同一个内在节奏时，整个界面会产生一种无法被指认、却能被感知的和谐。

就像一座好的建筑，你不会去数它的柱间距，但你会觉得走在里面很舒服。

我们也相信，工艺感不来自单个组件的精致，而来自系统的一致。

一个按钮好看，这不难。难的是：所有按钮、所有开关、所有卡片、所有弹窗、所有滑块、所有输入框，它们的阴影来自同一个光源，它们的圆角遵循同一个公式，它们的按压反馈拥有同一个时间函数，它们的禁用态呈现同一种褪色的温柔——这才是一个设计系统的真正气质。气质不是某个局部的亮眼，气质是整体的不可替代。

所以我们把每一个参数都写成了令牌。

颜色、阴影、间距、圆角、时长、缓动曲线——全部收敛在一份 tokens.css 文件里，不允许硬编码，不允许例外。这不是洁癖，这是让 AI 代码助手也能精准地生成属于这个系统的界面的唯一方法。当机器能够理解规则，人类就能专注于感受。

Tactile UI 不追逐潮流。

潮流会告诉你今年流行毛玻璃、明年流行新拟物、后年流行某种我们还不知道名字的风格。我们不参与这场轮换。我们只做一件事：让屏幕上的物体看起来可以触碰，让触碰的瞬间有一个微小的、确切的、不可替代的回应。

这件事不会过时。

因为手指的记忆比眼睛的审美更长久。你可能忘记某个 App 的配色方案，但你会记得那个开关拨过去时、恰到好处的"咔"。

---

*Tactile UI。*
*让屏幕重新拥有手感。*

---

*它不是对 Smartisan OS 的复刻，也不是简单的拟物风组件库。*
*它提炼的是背后的设计方法：秩序、比例、触感、直觉、克制的工艺感。*
*以及，可被 AI Coding Agent 理解和执行的设计规则。*
